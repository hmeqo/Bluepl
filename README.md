# Bluepl

## Build

Make a sender.py file on /src/email/, contents like

```python
# /src/email/sender.py
host = "smtp.email.com"
port = 25
sender = "123456789@email.com"
password = "123456789"
```

cd and run command on Bluple-web

```bash
python build.py
```

## Run

run on main directory

```bash
python main_windows.py
```
